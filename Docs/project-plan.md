# DNS Monitoring Dashboard

## Objective

Build a web-based cybersecurity dashboard that monitors DNS traffic and detects suspicious domains.

## Technologies

### Frontend

- React
- Tailwind CSS

### Backend

- Python
- FastAPI

### Database

- PostgreSQL

### Networking

- DNS
- TCP/IP
- UDP

### Security

- DNS Analysis
- Threat Detection

## Features

- DNS packet capture
- DNS query storage
- Dashboard analytics
- Suspicious domain detection
- DNS tunneling detection
- Interactive charts

## Deployment

Frontend:

- Vercel

Backend:

- Render

Database:

- Neon PostgreSQL
