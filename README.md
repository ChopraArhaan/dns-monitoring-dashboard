# dns-monitoring-dashboard
A web-based DNS monitoring and threat detection dashboard built using FastAPI, React, PostgreSQL, and Scapy.
# DNS Monitoring & Visualization Dashboard

A real-time cybersecurity dashboard that captures DNS traffic,
analyzes queries, identifies suspicious domains, and visualizes
network activity through interactive charts.

## Features

- DNS Query Monitoring
- Suspicious Domain Detection
- DNS Tunneling Detection
- Real-Time Analytics
- Interactive Dashboard
- Threat Intelligence Integration

## Technologies

Frontend:
- React
- Tailwind CSS
- Chart.js

Backend:
- FastAPI
- Python
- Scapy

Database:
- PostgreSQL

Deployment:
- Vercel
- Render

## Architecture

Collector Agent
      ↓
FastAPI Backend
      ↓
PostgreSQL
      ↓
React Dashboard
